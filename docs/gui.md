# NTSCloom GUI Usage

## Import
- Click **Import Video or Image** to load media.
- Images (PNG/JPG) are supported in this prototype. Video imports surface a helpful message.

## Preview
- Realtime preview updates the processed output for fast iteration.
- Full preview uses the full-resolution input.

## Presets
- Use the Preset dropdown to switch factory presets.
- Save custom presets to JSON or load them from disk.

## Render / Export
- Click **Render / Export** to process the full-resolution frame.
- Choose PNG or JPEG format in the save dialog.

## Tips
- Use the right panel sliders for composite, channel, and tape controls.
- Hover over controls for parameter tooltips.
