# NTSCloom UI / UX

## Layout

- **Left dock**: Import, Presets, Project.
- **Center**: Preview with scrubbing and playback controls.
- **Right dock**: Controls grouped into sections:
  - Composite Encoding
  - Channel Filters
  - Chroma
  - Luma
  - Tape/VHS
  - Noise & Interference
  - Temporal
  - Output

## Preview

- Quality modes: realtime (low-res) and full-quality render.
- Optional scopes: waveform, vectorscope, YCbCr view.

## Export

- Render queue with presets.
- Formats: ProRes, H.264, H.265.
- Bit depth: 8/10/12-bit, optional alpha preservation.

## Presets

- NTSC-TV, VCR-Normal, VCR-BadTape, Broadcast-Noisy, Vintage-Posterized.
- User presets stored as JSON, shareable.
